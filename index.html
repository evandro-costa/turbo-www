High performance Lua web and networking toolkit, that's modern, fresh, object oriented and easy to modify. Written in pure Lua. Turbo.lua is non-blocking and features a fast web server and a HTTP client. The toolkit is good for REST APIs, traditional HTTP requests and open connections like Websockets and offers a high degree of freedom to do whatever you want, your way.

Download v1.0 Beta API docs View Github

Why?

Intuitive

  • Using the native Lua coroutine to enable async operations
  • Offers callback based event driven programming
  • Single thread, no locking required

Fast

  • Native speed thanks to the fastest tracing JIT available
  • Utilizes the LuaJIT FFI, syscalls are done directly in Lua
  • Uses the handwritten HTTP parser by Ryan Dahl
  • Low-level operations available when needed

Easy to use, easy to customize

  • Modeled after the widely used Tornado Web API
  • All code is in one language and available for everyone to study and modify
  • Fully object-oriented

Hello World

Making a web server that responds with the canonical message is easy. 

				
local turbo = require "turbo"
local ExampleHandler = class("ExampleHandler", turbo.web.RequestHandler)
function ExampleHandler:get()
	self:write("Hello world!")
end

local application = turbo.web.Application({{"^/$", ExampleHandler}}):listen(8888)

turbo.ioloop.instance():start()
